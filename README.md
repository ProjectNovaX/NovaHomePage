## Terminal Commands

1. Download and Install NodeJs LTS version from [NodeJs Official Page](https://nodejs.org/en/download/).
2. Clone repository.
3. Navigate to the root ./ directory and run `npm install` to install our local dependencies.
4. Run `npm start`.
